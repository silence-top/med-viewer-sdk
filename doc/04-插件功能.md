## 工具栏（Toolbar）

悬浮工具栏，内置标注、颜色调整、截图、旋转/翻转等预设按钮，支持完全自定义按钮组。

### 基础配置

```ts
plugins: {
  toolbar: true   // 使用默认配置
}

// 或传入 ToolbarOptions
plugins: {
  toolbar: {
    position: 'BOTTOM_RIGHT',  // 工具栏位置，默认 MIDDLE_LEFT
    buttons: [...]              // 自定义按钮，不传则使用内置预设
  }
}
```

### ToolbarOptions

```ts
interface ToolbarOptions {
  position?: ToolbarPosition   // 工具栏停靠位置
  buttons?: ToolbarButton[]    // 自定义按钮列表（覆盖预设）
}
```

### 位置枚举（ToolbarPosition）

`TOP_LEFT` / `TOP_CENTER` / `TOP_RIGHT` / `BOTTOM_LEFT` / `BOTTOM_CENTER` / `BOTTOM_RIGHT` / `MIDDLE_LEFT` / `MIDDLE_RIGHT`

### 内置预设按钮

| 按钮 ID | 功能 | 前置条件 |
|--------|------|---------|
| `anno` | 标注工具（矩形/多边形/圆形/椭圆/线段/手绘） + 颜色选择 | 需启用 `annotorious` |
| `colorAdjust` | 颜色调整（亮度/对比度/饱和度/伽马/色相/反色/怀旧/灰度） + 旋转翻转 | 需启用 `colorAdjust` |
| `selection` | 区域截图工具 | 需启用 `selection` |

### 自定义按钮

```ts
import { type ToolbarButton } from 'med-viewer-sdk'

const customBtn: ToolbarButton = {
  id: 'my-btn',
  icon: 'https://example.com/icon.png',  // 图标 URL
  label: '我的工具',
  onClick: (engine, hide) => {
    console.log('点击了自定义按钮', engine)
    // hide()  // 可以在这里关闭下拉框
  }
}

// 带下拉面板的按钮
const dropdownBtn: ToolbarButton = {
  id: 'my-dropdown',
  icon: 'data:image/png;base64,...',
  label: '更多',
  activeOnDropdown: true,  // 下拉打开时自动激活该按钮
  dropdownContent: (engine, hide) => {
    const div = document.createElement('div')
    div.innerHTML = '<button>操作1</button>'
    div.querySelector('button')?.addEventListener('click', () => {
      // 执行操作
      hide()  // 关闭下拉
    })
    return div
  }
}

// 带外部事件监听的激活状态按钮
const activeEventBtn: ToolbarButton = {
  id: 'anno-btn',
  icon: '...',
  activeEvent: {
    emitter: engine.anno,    // 监听 anno 的事件来更新激活态
    event: 'modeChange',
    mapToActive: (data) => data.enabled   // 返回 true/false 控制激活态
  }
}
```

### 运行时控制

```ts
// Toolbar 实例挂载在 engine.toolbar
engine.toolbar?.destroy()  // 销毁工具栏
```

## 标注（Annotorious）

基于 [Annotorious v2.7.17](https://annotorious.github.io/) 的封装，提供 6 种矢量标注工具，支持 W3C 标注数据格式。

### 配置

```ts
plugins: {
  annotorious: true  // 基础启用

  // 或传入完整配置
  annotorious: {
    locale: 'zh-CN',
    onCreateAnnotation: (annotation) => {
      console.log('新建标注', annotation)
    },
    onUpdateAnnotation: (annotation, previous) => {
      console.log('更新标注', annotation)
    },
    onDeleteAnnotation: (annotation) => {
      console.log('删除标注', annotation)
    },
    options: {
      // 测量标签配置（可选）
      formatter: {
        pixelsPerMeter: 250000,  // 每米像素数，用于换算实际尺寸
        showMeasure: true        // 是否在标注上显示测量值
      }
    }
  }
}
```

### AnnotoriousOptions

```ts
interface AnnotoriousOptions {
  locale?: string                         // 语言，默认同引擎 locale
  onCreateAnnotation?: (anno: any) => void
  onUpdateAnnotation?: (anno: any, prev: any) => void
  onDeleteAnnotation?: (anno: any) => void
  onCancelAnnotation?: (anno: any) => void
  options?: {
    formatter?: {
      pixelsPerMeter?: number   // 物理像素密度，用于测量换算
      showMeasure?: boolean     // 是否显示测量信息
    }
  }
  [key: string]: any   // 其余选项直接透传给 Annotorious
}
```

### API（通过 engine.anno 访问）

```ts
const anno = engine.anno!

// 设置绘图工具
anno.setTool('rect')       // 矩形
anno.setTool('polygon')    // 多边形
anno.setTool('circle')     // 圆形（以中心点+半径绘制）
anno.setTool('ellipse')    // 椭圆
anno.setTool('line')       // 线段
anno.setTool('freehand')   // 手绘
anno.setTool('rect', '#ff0000')  // 带颜色

// 启用/禁用绘制模式
anno.setEnabled(true)
anno.setEnabled(false)

// 标注数据管理
anno.getAnnotations()          // 返回 W3C 格式的标注数组
anno.setAnnotations(data)      // 批量设置标注（覆盖当前）
anno.clear()                   // 清除全部标注

// 语言切换
anno.setLocale('en-US')

// 事件监听（外部扩展）
anno.on('modeChange', ({ enabled, tool }) => {
  console.log('绘图模式变化', enabled, tool)
})
```

### 标注数据格式（W3C Web Annotation）

```json
{
  "@context": "http://www.w3.org/ns/anno.jsonld",
  "type": "Annotation",
  "id": "#uuid-xxx",
  "body": [{ "type": "TextualBody", "value": "备注内容", "purpose": "commenting" }],
  "target": {
    "source": "http://your-image-uri",
    "selector": {
      "type": "SvgSelector",
      "value": "<svg>...</svg>"
    }
  }
}
```

### 功能边界

- 版本锁定 Annotorious **v2.7.17**（`annotorious-openseadragon-ld`）
- 不支持文字标注工具（Annotorious v2 限制）
- 标注持久化（保存到服务器）需业务层自行实现：在 `onCreateAnnotation`/`onUpdateAnnotation` 回调中调用接口
- 测量换算功能依赖 `pixelsPerMeter` 精度，需与扫描仪参数匹配

## 颜色调整（ColorAdjust）

基于 Canvas 2D 像素操作的图像滤镜，使用 WebGL 渲染管线（通过 `openseadragon-filtering`），支持实时调整。

> 需要浏览器支持 **WebGL**，否则颜色调整不生效。

### 配置

```ts
plugins: {
  colorAdjust: true  // 使用默认值（全部参数归零/中性）

  // 或传入初始值
  colorAdjust: {
    adjustments: {
      brightness: 1.2,    // 亮度，默认 1.0，范围建议 0.1~3.0
      contrast: 1.0,      // 对比度，默认 1.0
      saturation: 1.0,    // 饱和度，默认 1.0，0=灰度
      gamma: 1.0,         // 伽马，默认 1.0
      hue: 0,             // 色相旋转角度，默认 0，范围 -180~180
      invert: false,      // 反色
      sepia: false,       // 怀旧
      greyscale: false    // 灰度
    },
    debounceMs: 60,                      // 防抖延迟（毫秒），默认 60
    loadMode: 'async',                   // 渲染模式：'async'（推荐）/ 'sync'
    onAdjustmentsChange: (adj) => { ... } // 调整回调
  }
}
```

### API（通过 engine.colorAdjust 访问）

```ts
const ca = engine.colorAdjust!

// 更新调整参数（支持局部更新，内置防抖）
ca.setAdjustments({ brightness: 1.5, invert: true })

// 读取当前参数
console.log(ca.adjustments)  // ColorAdjustments

// 重置为初始配置
ca.reset()
```

### ColorAdjustments

```ts
interface ColorAdjustments {
  brightness?: number    // 亮度乘数，1.0 = 无变化
  contrast?: number      // 对比度乘数，1.0 = 无变化
  saturation?: number    // 饱和度乘数，0 = 纯灰度，1.0 = 原始
  gamma?: number         // 伽马校正，1.0 = 无变化
  hue?: number           // 色相旋转（度），0 = 无变化
  invert?: boolean       // 反色
  sepia?: boolean        // 怀旧效果（优先级高于 hue/saturation）
  greyscale?: boolean    // 灰度（优先级最高）
}
```

### 性能说明

- 使用 **预计算 LUT（256 色查找表）** 将亮度、对比度、反色、伽马合并为单次循环，减少像素遍历开销
- `setAdjustments()` 内置 `debounceMs`（默认 60ms）防抖，适合滑块实时拖拽
- `loadMode: 'async'`（推荐）将渲染推入宏任务队列，保持平移/缩放流畅

## 区域截图（Selection）

提供可拖拽的矩形选区工具，支持旋转，确认后回调选区坐标及截图 Blob。

### 配置

```ts
plugins: {
  selection: true  // 启用，使用默认配置

  selection: {
    showSelectionControl: true,   // 显示切换按钮
    showConfirmDenyButtons: true, // 显示确认/取消按钮
    returnPixelCoordinates: true, // 回调使用像素坐标（false=视口坐标）
    keyboardShortcut: 'c',        // 键盘快捷键，默认 'c'
    allowRotation: true,          // 允许旋转选区
    restrictToImage: false,       // 是否限制在图像范围内
    onSelection: (rect, blob) => {
      // rect: OpenSeadragon.Rect（图像像素坐标）
      // blob: Blob | null（截图数据，Canvas 需 CORS 允许）
      console.log(rect.x, rect.y, rect.width, rect.height)
      if (blob) {
        const url = URL.createObjectURL(blob)
        // 下载或展示截图
      }
    },
    onSelectionCanceled: () => {
      console.log('选区已取消')
    },
    onSelectionChange: (rect) => {
      console.log('选区调整中', rect)
    },
    onSelectionToggled: ({ enabled }) => {
      console.log('选区工具状态', enabled)
    }
  }
}
```

### API（通过 engine.selection 访问）

```ts
const sel = engine.selection!

// 启用/禁用选区工具
sel.enable()
sel.disable()

// 更新国际化文字
sel.updateLocale()
```

### 功能边界

- 截图 Blob 依赖 Canvas 跨域权限（图像服务需返回正确的 CORS 头），否则 `blob` 为 `null`
- 旋转选区的坐标为旋转后的外接矩形
- 快捷键 `c` 可快速进入/退出选区模式

## 比例尺（Scalebar）

在 Viewer 上叠加物理比例尺，支持显微镜和地图两种显示模式。

### 配置

```ts
plugins: {
  scalebar: true  // 默认 MAP 模式，停靠 BOTTOM_LEFT

  scalebar: {
    type: ScalebarType.MICROSCOPY,  // 或 ScalebarType.MAP
    pixelsPerMeter: 250000,         // 每米像素数（决定比例尺物理精度）
    location: ScalebarLocation.BOTTOM_RIGHT,
    xOffset: 10,
    yOffset: 10,
    color: 'white',
    fontColor: 'white',
    backgroundColor: 'rgba(0,0,0,0.5)',
    fontSize: '12px',
    fontFamily: 'Arial',
    barThickness: 3,
    minWidth: '100px'
  }
}
```

### ScalebarOptions

```ts
interface ScalebarOptions {
  type?: ScalebarType          // 显示模式
  pixelsPerMeter?: number      // 每米像素数，影响比例尺精度
  xOffset?: number             // 水平偏移（px）
  yOffset?: number             // 垂直偏移（px）
  stayInsideImage?: boolean    // 是否限制在图像范围内
  color?: string               // 比例尺颜色
  fontColor?: string           // 文字颜色
  backgroundColor?: string     // 背景色
  fontSize?: string            // 字体大小
  fontFamily?: string          // 字体
  barThickness?: number        // 比例尺厚度（px）
  minWidth?: string            // 最小宽度（CSS 值）
  location?: ScalebarLocation  // 停靠位置
  sizeAndTextRenderer?: (ppm: number, minSize: number) => { size: number; text: string }
}
```

### ScalebarType 枚举

| 值 | 说明 |
|----|------|
| `NONE (0)` | 不显示比例尺 |
| `MICROSCOPY (1)` | 显微镜模式（µm / mm） |
| `MAP (2)` | 地图模式（m / km） |

### ScalebarLocation 枚举

`NONE(0)` / `TOP_LEFT(1)` / `TOP_RIGHT(2)` / `BOTTOM_RIGHT(3)` / `BOTTOM_LEFT(4)`

### 功能边界

- 比例尺精度完全依赖 `pixelsPerMeter` 参数的准确性
- 枚举值支持字符串传入（如 `type: 'MICROSCOPY'`），SDK 内部自动转换

## 倍率显示（Magnification）

在 Viewer 上显示当前缩放倍率，并提供固定倍率快捷按钮（4X / 10X / 20X / 40X / Fit）。

### 配置

```ts
plugins: {
  magnification: true  // 默认 OSD 模式，停靠 MIDDLE_LEFT

  magnification: {
    type: MagnificationType.LD,              // 或 MagnificationType.OSD
    position: MagnificationPosition.BOTTOM_LEFT,
    offsetX: 0,
    offsetY: 0,
    pixelsPerMeter: 250000                   // LD 模式下必须提供
  }
}
```

### MagnificationOptions

```ts
interface MagnificationOptions {
  type?: MagnificationType           // 倍率计算模式
  position?: MagnificationPosition   // 控件停靠位置
  offsetX?: number                   // 水平偏移（px）
  offsetY?: number                   // 垂直偏移（px）
  pixelsPerMeter?: number            // LD 模式专用，物理像素密度
}
```

### MagnificationType 枚举

| 值 | 适用场景 | 说明 |
|----|---------|------|
| `OSD` | 标准 DZI / IIIF | 从图像源 `max_magnification` 字段读取基准倍率 |
| `LD` | 联影私有协议切片 | 通过 `pixelsPerMeter` + 固定公式换算，倍率按钮同步触发 `goToPosition` |

### 功能边界

- OSD 模式下图像源须包含 `max_magnification` 元数据字段，否则倍率显示为 `-`
- LD 模式倍率按钮依赖 `pixelsPerMeter` 精度
