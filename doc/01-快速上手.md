# 快速上手

## 环境要求

| 依赖 | 版本 | 说明 |
|------|------|------|
| Node.js | ≥ 22.x | 推荐使用 Volta 管理版本 |
| openseadragon | ^3.1.0 | Peer Dependency，需手动安装 |
| vue | ≥ 2.7.0 或 ≥ 3.0.0 | Peer Dependency，仅在 Vue 集成时需要 |
| 浏览器 | Chrome 80+ / Firefox 80+ / Safari 14+ / Edge 80+ | 颜色调整功能需要 WebGL |

## 安装

```bash
# 安装 SDK
npm install med-viewer-sdk

# 安装 Peer Dependencies
npm install openseadragon@^3.1.0
```

如需 Vue 集成：
```bash
npm install vue@^3.0.0   # 或 vue@^2.7.0
```

## 30 秒上手

### 原生 HTML / JS

```html
<div id="viewer" style="width:800px;height:600px"></div>

<script type="module">
import { MedViewerEngine } from 'med-viewer-sdk'

const engine = new MedViewerEngine({
  osdOptions: {
    id: 'viewer',
    tileSources: 'https://your-server/iiif/manifest.json'
  },
  locale: 'zh-CN',
  theme: 'default',
  plugins: {
    toolbar: true,
    selection: true,
    colorAdjust: true,
    scalebar: true,
    magnification: true,
    annotorious: true
  }
})

window.engine = engine  // 方便调试
</script>
```

### Vue 3

```vue
<template>
  <MedViewer :options="viewerOptions" theme="light" @ready="onReady" />
</template>

<script setup>
import { MedViewer } from 'med-viewer-sdk'

const viewerOptions = {
  osdOptions: {
    id: 'viewer',
    tileSources: 'https://your-server/iiif/manifest.json'
  },
  plugins: { toolbar: true }
}

const onReady = (engine) => {
  console.log('查看器已就绪', engine)
}
</script>
```

### UMD 方式

```html
<script src="https://unpkg.com/openseadragon@3/build/openseadragon/openseadragon.min.js"></script>
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
<script src="node_modules/med-viewer-sdk/dist/med-viewer-sdk.umd.js"></script>

<script>
const { MedViewerEngine } = window.MedViewerSDK
const engine = new MedViewerEngine({ osdOptions: { id: 'viewer', tileSources: '...' } })
</script>
```

## 构建产物

| 文件 | 格式 | 用途 |
|------|------|------|
| `dist/med-viewer-sdk.mjs` | ESM | Vite / Webpack 等现代打包工具 |
| `dist/med-viewer-sdk.umd.js` | UMD | 原生 `<script>` 引入、CDN |
| `dist/med-viewer-sdk.d.ts` | TypeScript 声明 | IDE 类型提示 |
| `dist/style.css` | CSS | 样式文件（需手动引入） |
